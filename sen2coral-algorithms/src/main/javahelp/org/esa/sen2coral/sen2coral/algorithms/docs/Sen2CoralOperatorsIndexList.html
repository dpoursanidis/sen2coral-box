Sen2Coral Processing

Sen2Coral Algorithms

Name

Short Description

Deglint
The Deglint processor uses NIR or SWIR bands to remove glint from visible bands.

Depth Invariant Indices
The Depth Invariant Indices processor produces depth invariant bands from pairs of image bands. Image must contain identifiable areas of sand at different depths.

Empirical Bathymetry
The Empirical Bathymetry processor maps bathymetry based on regression of log band ratio (Lyzenga method). Ancillary data required: Calibration data set of points in the image of known bathymetry.

Land, Cloud and White Cap Mask
The Land, cloud and white cap mask processor produces a mask layer for land, clouds and white caps.

Radiometric Normalisation
The Radiometric Normalisation processor radiometrically aligns two or more images based on control points over pseudo-invariant features (PIFs).
